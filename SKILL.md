# 文档格式转换工具 (Doc Formatter) SKILL

## 🎯 工具概述

这是一个专门为大学生设计的文档格式转换工具，可以帮助你：
- 将格式混乱的Word文档转换为规范的作业格式
- 自动修复常见的格式问题（标题层级、列表格式、加粗文本等）
- 支持Markdown和Word两种输出格式
- 批量处理多个文档

## 🚀 快速开始

### 基本使用流程

1. **准备文档**：将需要格式化的Word文档放在 `memory/` 目录下
2. **调用工具**：告诉我文档路径和需要的输出格式
3. **获取结果**：工具会生成格式规范的文档

### 示例命令

```
# 转换单个文档
请帮我格式化 memory/没有格式.docx 文档

# 批量转换所有文档  
请帮我批量处理 memory/ 目录下的所有Word文档

# 指定输出格式
请将文档转换为Markdown格式并保存
```

## 🔧 核心功能

### 1. 格式自动修复
- **标题层级**：自动识别并调整标题层级（H1-H4）
- **列表格式**：修复无序列表和有序列表的格式
- **加粗文本**：统一加粗文本的Markdown语法
- **空格清理**：移除多余的空格和空行

### 2. 输出格式支持
- **Word文档**：生成可以直接提交的 `.docx` 文件
- **Markdown**：生成便于版本控制的 `.md` 文件
- **双重输出**：同时生成两种格式的文件

### 3. 批量处理
- 支持同时处理多个文档
- 保持所有文档格式一致
- 生成统一的文件命名规范

## 📁 文件结构

处理后的文件会保存在 `memory/` 目录下，命名规则：
- `格式化后的_[原文件名].md` - Markdown版本
- `格式化后的_[原文件名].docx` - Word版本
- `作业提交_[原文件名].docx` - 可直接提交的版本

## ⚠️ 注意事项

1. **依赖安装**：首次使用需要安装 `python-docx` 库
2. **中文支持**：默认使用"微软雅黑"字体，确保中文显示正常
3. **文件大小**：建议文档不超过10MB
4. **复杂格式**：对于特别复杂的格式（如表格、图片），可能需要手动调整

## 🎨 高级功能

### 自定义模板
你可以提供自定义的格式模板，工具会按照模板调整文档格式。

### 格式检查
工具会生成格式检查报告，指出所有修复的格式问题。

### 批量重命名
支持按照学号、课程名等规则批量重命名文件。

## 🔍 故障排除

### 常见问题

1. **"ModuleNotFoundError: No module named 'docx'"**
   - 解决方案：运行 `pip install python-docx`

2. **中文乱码**
   - 解决方案：确保文档使用UTF-8编码

3. **格式丢失**
   - 解决方案：复杂格式可能需要手动调整，或提供原始文档查看

### 调试模式
如果需要查看详细的处理过程，可以要求开启调试模式。
