# 📚 文档格式转换工具使用示例

## 🎯 场景描述

小明是一名大学生，他需要提交多个课程作业，但是他的Word文档格式混乱：
- 标题层级不正确
- 列表格式不统一
- 加粗文本格式错误
- 需要转换为规范的作业格式

## 📋 原始文档示例

假设小明的原始文档 `作业1_混乱格式.docx` 内容如下：

```
### 用例描述：处理检测记录（生成车速检测记录）
---
#### 用例名称
处理检测记录（生成车速检测记录）
#### 概述
车速检测器将检测到的车辆信息（车牌号、时间、车速）发送至后台管理系统
#### 依赖
- **包含关系** :包含「警示短信发送」用例
1.车速检测器已正常部署
2. 后台管理系统已配置
```

## 🚀 使用流程

### 步骤1：准备文档
小明将文档放在 `memory/` 目录下：
```
memory/
  ├── 作业1_混乱格式.docx
  ├── 作业2_实验报告.docx
  └── 作业3_课程论文.docx
```

### 步骤2：调用工具
小明对AI说：
```
请帮我格式化 memory/作业1_混乱格式.docx 文档
```

### 步骤3：AI处理
AI会：
1. 读取文档内容
2. 分析格式问题
3. 转换为规范格式
4. 生成输出文件

### 步骤4：获取结果
AI返回：
```
✅ 文档格式化完成！

📋 处理结果：
- 输入文件: memory/作业1_混乱格式.docx
- 发现格式问题: 3个
- 生成文件:
  ✅ Markdown文件: memory/格式化后的_作业1_混乱格式.md
  ✅ Word文档: memory/作业提交_作业1_混乱格式.docx

🔧 修复的格式问题：
1. 第4行：标题层级 - 将"#### 用例名称"改为"## 用例名称"
2. 第8行：列表格式 - 修复加粗列表项格式
3. 第9行：编号格式 - 修复编号后缺少空格
```

## 🔄 批量处理

如果小明有多个文档需要处理：

```
请帮我批量处理 memory/ 目录下的所有Word文档
```

AI会：
1. 查找所有 `.docx` 和 `.doc` 文件
2. 逐个进行格式化
3. 为每个文档生成规范版本
4. 提供处理汇总报告

## 💡 使用技巧

### 技巧1：预处理文档
- 确保文档使用常见字体
- 移除不必要的格式（如背景色、复杂边框）
- 将图片保存为独立文件

### 技巧2：分批处理
- 对于大量文档，建议分批处理（每次5-10个）
- 先测试一个文档，确认效果后再批量处理

### 技巧3：检查结果
- 总是检查生成的文档
- 对于重要文档，建议手动复核
- 保留原始文档作为备份
